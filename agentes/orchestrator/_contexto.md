# Orchestrator · Contexto predeterminado

## Horario de trabajo NosVers
- Angel disponible: fines de semana + noches de semana
- África disponible: ~2-4h/semana, horario flexible
- Publicaciones WordPress: solo con aprobación de Angel

## Prioridades permanentes
1. Seguridad: verificar tokens y credenciales cada semana
2. Vault: hacer backup a GitHub cada domingo
3. Bot Telegram: verificar que responde cada hora
4. Agentes: si un agente lleva +48h sin correr → alertar a Angel

## Alertas automáticas a Angel
- VPS: disco >80% usado
- Bot Telegram: sin respuesta >2h
- WordPress: nosvers.com caído
- Agente fallando 3 veces seguidas
