# Soil Food Web — Dr. Elaine Ingham

## Referencia principal
Doctora Elaine Ingham — Rodale Institute / Soil Foodweb Inc.
Sitio: soilfoodweb.com

## Concepto central
La red alimentaria del suelo es el conjunto de organismos vivos
que habitan el suelo y las relaciones que tienen entre ellos.

## Organismos clave
- Bacterias: descomponen materia orgánica, fijan nitrógeno
- Hongos: red de transporte de nutrientes, simbiosis con raíces (micorriza)
- Protozoarios: comen bacterias, liberan nutrientes en forma asimilable
- Nematodos: regulan poblaciones bacterianas y fúngicas
- Artrópodos: fragmentan materia orgánica, aerean el suelo
- Lombrics (Eisenia fetida): mezclan capas, aceleran descomposición

## LombriThé — base científica NosVers
Proceso aeróbico 24-48h con:
- Lombricompost maduro (base de microorganismos)
- Mélasse (alimento para bacterias)
- Protéine de poisson (nitrógeno para hongos)
- Farine de varech (minerales + hormonas vegetales)
- Aireación constante (bulleur)

Resultado: multiplicación 100-1000× de microorganismos benéficos
Diferencia con extracto simple: el thé MULTIPLICA vida, el extracto solo DILUYE
